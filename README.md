# Labs for Introduction to algorithm (2024 Sep.~Dec)
## How labs proceeded
1. Downloads questions from online judge
2. Solve it with the programming language you like
3. Upload to online judge, the code will be graded base on the correctness and speed.
## Lab0
The purpose of this lab is for students to get familiar to the judge.
## Lab1 (Some files are missing in this folder)
The problems are relating to sorting and binary serach. 
## Lab2
This lab is mainly focus on divide and conquer. Including fast matrix multiplication, shortest pairs of points, etc. 
## LAB3 
In this lab, graph-related algorithm is included. For instance, DAG with topological sort, dijkstra algorithm, and so on. 
## Lab4
Dynamic programming and greedy algorithm is the main scope for this lab. We implemented longest common subsequence, knapsnak problem, etc. 
## Lab5
This is the lab for us to get prepared for the final computer-based test. It include every subject in this semester.
